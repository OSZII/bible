# bible
Collection of Bible books as text files, with one chapter per file and one
verse per line. Within each chapter file, the line number is the verse number.
Blank lines are kept where the source marks omitted or bridged verses, so later
line numbers continue to match verse numbers.

Verse splits are generated from verse-marked source text, not inferred from
punctuation. The original 66-book corpus was the New King James Bible. Eastern
Orthodox / deuterocanonical material has been added from the public-domain
World English Bible Updated ecumenical edition from eBible.org. Esther and
Daniel now use the WEB Updated Greek/deuterocanonical forms so their additions
are included in the same chapter-per-file layout. The numbered directories
place this added Old Testament material before the New Testament.

Localized folders are generated by `scripts/build_localized_translations.py`
from verse-marked sources. Current generated folders are:

- `sr`: Serbian Synodal/SPC web text from Svetosavlje.
- `de`: Luther 1912 with Apocrypha OSIS fallback.
- `bg`: Bulgarian public-domain OSIS from `seven1m/open-bibles`.
- `el`: Greek Septuagint OT from eBible `grclxx` plus Patriarchal 1904 NT
  from eBible `grcbyz`.
- `es`: Spanish `spablm` from eBible.
- `fr`: French `frasbl` from eBible.
- `it`: Italian Riveduta 1927 from eBible `ita1927`.
- `ro`: Romanian `ronbtf` from eBible.
- `ru`: Russian Synodal from eBible `russyn`.
- `uk`: Ukrainian Freedom Bible from eBible `ukrfb`.
- `be`: Belarusian Antoni Boken translation from eBible `bel`
  (`CC BY-NC 4.0` terms in the source package).

Where a localized source lacks a project book or Orthodox addition, the chapter
file is still emitted with blank verse lines so line numbers stay aligned with
the English reference layout.

Georgian (`ka`) and Macedonian (`mk`) still need redistributable, verse-marked
source files before folders should be generated.
